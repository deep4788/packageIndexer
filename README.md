# Package Indexer

`pind`: A package indexer system to keeps track of package dependencies.

## Installation

```bash
$ cd packageIndexer
$ python setup.py install
```

## Usage

### Run Package Indexer Server

```bash
# Runs the package indexer; by default it runs on port 8080
$ pind

# Prints help message
$ pind -h
```
